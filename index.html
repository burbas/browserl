browserl by svahne

An Erlang Emulator written in javascript

Dependencies

Tested in Chromium 15 (fastest, loads to shell prompt in less than a second when hosted on same network), Firefox 3.6 and 7, Opera 11, Epiphany 3.2, Konqueror 4.7, iPhone 4 and on an HTC Wildfire running Android 2.2 (slowest)

Only limited support for binaries and ets tables have been added so far. Floats will not always work, especially not the presentation in the shell due to a dependency on binaries.

Only the most commonly used bifs have been implemented. Files and networking is not supported or expected to work, although some initial hacks have been made in order to get the test server running.

Is able to start Common test and the test server, and passes the standard OTP tests suites for lists, arrays, dicts, random numbers and queues.
